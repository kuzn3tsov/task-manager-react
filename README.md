## ğŸ¯ Usage Guide

### Creating Task Lists
1. Click the **"Create New List"** section in the sidebar
2. Enter a list name (e.g., "Work", "Personal", "Shopping")
3. Press Enter or click the â• button
4. Your new list becomes active automatically

### Adding Tasks
1. Select a task list from the sidebar
2. Fill out the task form:
   - **Title** (required): Brief description of the task
   - **Description** (optional): Detailed notes
   - **Priority**: Low, Medium, or High
   - **Due Date**: Set a deadline (optional)
3. Click **"Save"** or press Enter

### Managing Tasks
- **Complete Tasks**: Click the checkbox to mark as done
- **Edit Tasks**: Click on any task to edit its details
- **Delete Tasks**: Use the trash icon with confirmation
- **Filter Views**: Switch between Active and Completed tasks

### Setting Up Reminders
1. Click the **bell icon** ğŸ”” next to any list name
2. **Enable Email Reminders**:
   - Toggle Email Reminders ON
   - Enter your email address
   - Click "Test Email" to verify
3. **Enable SMS Reminders**:
   - Toggle SMS Reminders ON  
   - Enter your phone number
   - Click "Test SMS" to verify
4. **Configure Preferences**:
   - Choose reminder type (Daily Digest or When Due)
   - Select preferred time
   - Click "Save Reminders"

### Language Switching
- Use the **language dropdown** in the top-right corner
- Choose between **EN** (English) and **HR** (Croatian)
- Interface updates instantly with translations
